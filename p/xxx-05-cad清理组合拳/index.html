XXX-05-CAD清理组合拳

2025-12-22

如果你的图纸经常崩溃、显示异常，或者从厂家拷贝来的图块带了大量“脏数据”，这套流程就是你的救命稻草。这套组合拳的专业顺序通常是：Audit (体检) -> Overkill (瘦身) -> Purge (清扫)。

第一式：Audit（体检修复）

  • 功能： 扫描整个图纸的数据库，寻找并修复逻辑错误。
  • 应用场景： 图纸无故卡顿、复制粘贴失效、或者有些图形“看得见点不着”时。
  • 老手操作：
    • 输入 AUDIT。
    • 命令行询问“是否修复发现的任何错误？”，输入 Y（是）。
    • 注： 它会像杀毒软件一样扫描一遍，最后告诉你修复了多少个错误对象。

第二式：Overkill（重叠清理）

  • 功能： 自动删除重叠的线条、合并部分重叠的线。
  • 应用场景： 非常关键！尤其是当你准备把 AutoCAD 的 2D 草图导入 SolidWorks 建模时。SolidWorks 的草图最怕重叠线（会报错、无法拉伸），Overkill 能一键解决。
  • 老手设置：
    • 输入 OV。
    • 全选图纸。
    • 在弹出的对话框中，勾选**“忽略特性”**（比如颜色、线型）。这样即使一根红线和一根蓝线重叠，它也会把多余的那根删掉，只留下一根。

第三式：-Purge（深度强力清扫）

  • 功能： 除了普通 PU 能删掉的块和图层，这一招能删掉隐藏得更深的“注册应用程序”数据（Regapps）。
  • 现象： 有时候图纸里明明没东西，文件却有几十 MB 大，普通 PU 根本没用，这就是 Regapps 在作怪。
  • 老手操作：
    1. 输入 -PURGE（注意前面有个减号，这会启动命令行版本）。
    2. 输入 R（代表 Regapps）。
    3. 询问“要清理的名称”，直接按 Enter（默认全选）。
    4. 询问“是否确认每个要清理的名称”，输入 N（否）。
    • 奇迹时刻：你会看到命令行刷刷刷删掉成千上万个没用的注册项。

第四式：组合拳后的“收招”——REGEN

  • 功能： 重新生成图形（Regenerate）。
  • 作用： 所有的清理做完后，输入 RE，让 CAD 重新计算一次当前的显示数据。你会发现圆变圆了，线条缩放也顺滑了。
命令 昵称 解决的问题 对 SolidWorks 的意义
Audit 医生 修复数据库底层的“断裂”和错误。 保证文件导出时数据不损坏。
Overkill 裁缝 删除重叠线、多余的点、合并共线。 最高。 避免 3D 建模时草图闭合失败。
-Purge 清洁工 删除顽固的 Regapps 和无用块。 减小体积，防止 3D 软件读取文件时假死。

工作流

graph LR Start[获取外部厂家图纸] --> Step1[AUDIT: 逻辑体检修复] Step1 --> Step2[OV: 清理重叠几何体] Step2 --> Step3[PU: 清除无用块/图层] Step3 --> Step4[-PU > R: 强力清除 Regapps] Step4 --> Step5[RE: 重新生成显示] Step5 --> End[干净专业的工程模板]
